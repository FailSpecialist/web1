HTML

Hypertext Markup Language, short for Markup Language, is the Markup language used to create documents displayed on the browser screen.
- HTML allows users to split the screen and place text, images, hyperlinks, etc. using elements called 'tag'.
- Click Screen right-click in the browser to see HTML elements that make up the web page in the Elements tab in the developer tool.
You can also use 'F12' or 'Cmd+Option+I' (Mac) or 'Ctrl+Shift+I' (Windows) shortcuts. - Each tag is usually made up of a pair of starting tags and ending tags
- Each HTML tag can be assigned a 'style' property to define its size, alignment, font, color, etc.,
but CSS (Cascading Style Sheets) can be used to collect them in one place.
You can give elements the same properties.
